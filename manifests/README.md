# Manifests

This directory contains GDC manifests used to download whole-slide image data. Each manifest is logged below with the filters used
to generate it, the resulting size, and why that particular subset was chosen.

---

## gdc_manifest_full_luad_dx.txt

**Date generated:** 2026-06-23  
**Source:** [GDC Data Portal Repository](https://portal.gdc.cancer.gov/analysis_page?app=Downloads)  
**Result:** 478 cases, 541 files  

| Filter                 | Value            |
|------------------------|------------------|
| Program                | TCGA             |
| Project                | TCGA-LUAD        |
| Access                 | Open             |
| Data Format            | svs              |
| Data Type              | Slide Image      |
| Experimental Strategy  | Diagnostic Slide |

**Why:** Full open-access LUAD diagnostic (FFPE) slide cohort. Restricted to
Diagnostic Slide to exclude frozen Tissue Slide sections (`TS`/`BS` barcodes),
which carry ice-crystal artifacts and a domain gap from the FFPE data UNI/CONCH
were trained on. This is the base manifest all downstream subsets are sampled
from.

**Generated by:** manual filter selection in GDC Repository UI, no script.
